# mikrotik-css610-controller-api
Le switch MikroTik CSS610 tournant sous SwOS Lite ne possède pas d'interface CLI (SSH/Telnet). Ce projet permet d'automatiser sa configuration (VLANs, Reboot, Stats) via des requêtes HTTP en Python, permettant son intégration dans des pipelines DevOps.
